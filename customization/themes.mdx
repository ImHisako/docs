Illegalcord themes are CSS stylesheets that change the visual appearance of your Discord client — from color palettes and fonts to full interface overhauls. You can install themes from the online store, paste in your own CSS, or write a theme from scratch. Everything updates live, so you see changes the moment you apply them.

## What Themes Are

A theme is a `.css` file that Illegalcord injects into the Discord client after it loads. Because themes run as CSS, they can restyle any visible element — backgrounds, text colors, sidebars, buttons, and more — without touching Discord's underlying code or your account.

## Finding and Installing Themes

<Tabs>
  <Tab title="From the Store">
    The Illegalcord theme store hosts community-made themes that have been reviewed for safety.

    <Steps>
      <Step title="Open Discord Settings">
        Click the **gear icon** in the bottom-left corner of Discord to open User Settings.
      </Step>
      <Step title="Navigate to the Themes tab">
        Scroll down in the left sidebar until you see the **Illegalcord** section, then click **Themes**.
      </Step>
      <Step title="Open the store">
        Click the **Browse Themes** button at the top of the panel. The online theme store opens in a new overlay.
      </Step>
      <Step title="Install a theme">
        Find a theme you like and click the **Install** button on its card. Illegalcord downloads and registers it automatically.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Manual Installation">
    If you have a direct CSS URL or raw CSS code, you can install a theme without going through the store.

    <Steps>
      <Step title="Open the Themes tab">
        Go to **Discord Settings → Illegalcord → Themes**.
      </Step>
      <Step title="Open the Custom CSS editor">
        Click **Custom CSS** in the Themes panel.
      </Step>
      <Step title="Paste your CSS">
        Either paste a remote stylesheet URL using an `@import` rule, or paste raw CSS directly into the editor:

        ```css
        /* Import from a URL */
        @import url("https://example.com/my-theme.css");

        /* Or paste raw CSS directly */
        :root {
          --background-primary: #1a1a2e;
          --text-normal: #e0e0e0;
        }
        ```
      </Step>
      <Step title="Apply the theme">
        Click **Save & Apply**. Your changes take effect immediately.
      </Step>
    </Steps>

    <Warning>
      Themes from untrusted sources can contain malicious CSS — including styles that obscure UI elements or mimic phishing pages. Only install themes from authors you trust or from the official Illegalcord store.
    </Warning>
  </Tab>
</Tabs>

## Enabling and Disabling Themes

Every installed theme appears as a row in the **Themes** tab with a toggle switch on the right. Flip the toggle to enable or disable the theme instantly — no restart required.

You can have multiple themes enabled at the same time. Illegalcord stacks them in the order they appear in the list, so themes lower in the list can override rules from themes higher up.

<Tip>
  Drag theme rows to reorder them and control which styles take priority when rules conflict.
</Tip>

## Switching Between Multiple Themes

To swap your active look quickly, disable your current theme's toggle and enable another one. Because toggles are instant, you can preview themes back-to-back without any loading delay.

## Creating Your Own Theme

Building a theme requires only a basic understanding of CSS. Discord's interface uses CSS custom properties (variables) extensively, so overriding a handful of variables is often enough to produce a dramatic change.

<Steps>
  <Step title="Identify the elements you want to change">
    Open Discord's built-in DevTools by pressing <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>I</kbd> (or <kbd>Cmd</kbd>+<kbd>Option</kbd>+<kbd>I</kbd> on macOS). Use the element inspector to click on any part of the interface and see which CSS classes and variables it uses.
  </Step>
  <Step title="Create your CSS file">
    Create a new file with a `.css` extension and start by overriding CSS variables on `:root`:

    ```css
    /* my-theme.css */
    :root {
      --background-primary: #1e1e2e;
      --background-secondary: #181825;
      --text-normal: #cdd6f4;
      --text-muted: #a6adc8;
      --brand-experiment: #cba6f7;
    }
    ```
  </Step>
  <Step title="Target specific elements">
    For more precise changes, target Discord's generated class names directly. Because these names can change with Discord updates, prefer CSS variables where possible:

    ```css
    /* Style the member list sidebar */
    [class*="members_"] {
      background-color: #181825;
    }

    /* Round message avatars more aggressively */
    [class*="avatar_"] {
      border-radius: 50% !important;
    }
    ```
  </Step>
  <Step title="Import your theme into Illegalcord">
    Save the file, then paste its contents (or host it and use `@import`) into the **Custom CSS** editor under **Settings → Illegalcord → Themes**.
  </Step>
</Steps>

<Tip>
  Press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>I</kbd> inside Discord to open DevTools and inspect any element in real time. The **Styles** panel shows you exactly which CSS properties are active and where they come from.
</Tip>

## Popular CSS Variables

Override these variables in `:root` to cover the most visible parts of Discord's interface:

| Variable | Description |
|---|---|
| `--background-primary` | Main background color of the app (chat area, panels) |
| `--background-secondary` | Secondary background used for sidebars and headers |
| `--text-normal` | Default color for body text and message content |
| `--text-muted` | Dimmed text color for timestamps, hints, and placeholders |
| `--brand-experiment` | Discord's accent/brand color used for buttons and highlights |

<Note>
  Discord occasionally renames or removes CSS variables when it ships interface updates. If a variable stops working, use DevTools to find the current replacement.
</Note>
