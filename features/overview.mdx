## Full Feature List

The table below covers every major capability Illegalcord ships with out of the box. Some features require a specific plugin to be enabled — the **Notes** column calls those out.

| Feature | Description | Notes |
|---|---|---|
| **Themes** | Full CSS theme support — load community themes or write your own | Requires a `.css` or `.theme.css` file |
| **Plugins** | Extensible plugin system with enable/disable toggles per plugin | First-party and community plugins supported |
| **Message Tweaks** | Compact mode, configurable timestamp formats, and embed controls | Embed blocking requires the EmbedBlocker plugin |
| **Privacy** | Disable telemetry, hide typing indicators, suppress read receipts | NoTrack plugin required for analytics blocking |
| **Keybinds** | Custom keyboard shortcuts for any Discord action or Illegalcord toggle | Global keybinds require OS-level accessibility permissions |
| **Transparent Window** | Optional transparent client background that blends with your desktop | Works best paired with a matching CSS theme |
| **Auto-Update** | Silent background updates keep Illegalcord current without interrupting you | Can be disabled in Settings → General |
| **Cloud Sync** | Optionally sync your Illegalcord settings across multiple devices | Opt-in; requires an Illegalcord account |

<Info>
  Features marked as requiring a plugin are off by default. Head to **Settings → Plugins** to browse and enable them at any time.
</Info>
